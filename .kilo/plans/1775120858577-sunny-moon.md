# Pagination Template Edit Plan

## Goal
Edit the pagination template used in patient/billing.blade.php to customize the appearance or behavior of pagination links.

## Location of Template File
The pagination template referenced by `vendor.pagination.admin-portal` is located at:
```
resources/views/vendor/pagination/admin-portal.blade.php
```

## Current Template Content
The template contains a complete Bootstrap-based pagination implementation with:
- Previous/next page links
- Page number links
- Active page highlighting
- Responsive layout
- Custom styling via inline CSS

## How to Edit
To modify the pagination appearance or behavior:
1. Open `resources/views/vendor/pagination/admin-portal.blade.php`
2. Modify the HTML structure, CSS classes, or inline styles as needed
3. Changes will affect all views using this pagination template, including:
   - patient/billing.blade.php (the original query)
   - Any other views using `->links('vendor.pagination.admin-portal')`

## Example Modifications
- Change colors by modifying the `var(--admin-primary)`, `var(--admin-secondary)`, etc. CSS variables
- Adjust spacing, borders, or hover effects in the inline style section
- Modify the HTML structure for different layout requirements
- Change the pagination size or alignment by adjusting the container classes

## Verification
After making changes, verify the pagination appears correctly in:
- patient/billing.blade.php
- Any other pages using the same pagination template
